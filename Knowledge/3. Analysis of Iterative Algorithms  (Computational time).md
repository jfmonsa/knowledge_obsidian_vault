### 2- Ex. nested loop dependent indexes
```Python
def algo(n):
	s=0 # c1,1
	i=1 # c2,1
	while i<=n: # c3,n+1
		t=0 # c4,n
		j=1 # c5,n
		while j<=i: # c6,n(n+3)/2
			t +=1  # c7,n(n+1)/2
			j +=1  # # c8,n(n+1)/2
		s+=t # c9,n
		i+=1 # c10,n
```
